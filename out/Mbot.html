Class: Mbot

Mbot(modules, commands, ui)

Handles the bare minimum for mineflayer. Join and quit game. View bot and inventory in web browser. Can use UI to interact with bot. Other functionality is handled by other modules.

new Mbot(modules, commands, ui)

Parameters:

modules object An object with only instances of Module to mount onto the bot. The key "mbot" is reserved for this instance of Mbot.
commands Commands The commands that act like shortcuts to call instructions.
